소개

안녕하세요! 웹스토리보이입니다. 이 강의는 자바스크립트 초급부터 고급까지 아우르는 자바스크립트 핵심 개념을 다루는 기본서 강의입니다. 일반적인 자바스크립트 강의와는 목표와 방식이 다릅니다. 여기에서는 자바스크립트를 어떻게 단순하게 이해하고, 현실적으로 활용할 수 있는지를 고려하여 만들어진 내용입니다.

자바스크립트를 배우면서 때로는 이해하기 어려운 부분이나 무엇을 해야 하는지 혼란스러운 부분이 있을 수 있습니다. 그런 어려움을 극복하고 불필요한 시간 낭비를 줄이기 위해 이 강의는 차별화된 접근 방식을 채택하였습니다. 또한, 개인적으로도 어려움을 겪었던 부분들을 고려하여 설명하였습니다.

이 강의의 주요 목표는 자바스크립트를 학습하면서, 이론 뿐만 아니라 실제로 어떻게 활용할 수 있는지를 강조하는 것입니다. 이를 위해 강의는 크게 네 가지 주요 영역으로 구성되었습니다. 데이터 저장, 데이터 불러오기, 데이터 실행, 그리고 데이터 제어입니다. 이렇게 체계적으로 기초를 다지고 나서 실제 상황에 유용하게 활용할 수 있는 실용적인 예제를 다룰 것입니다. 이를 통해 프로그래밍 실력을 크게 향상시킬 수 있는 기회를 가지게 될 것입니다. 이 강의를 통해 자바스크립트의 세계에 더욱 가깝게 다가갈 수 있길 바랍니다.

1. 데이터 저장하기

자바스크립트의 가장 핵심적인 부분은 데이터를 저장하고 이를 출력하는 방법입니다. 이것이 자바스크립트의 기초이자 핵심입니다. 사실 모든 프로그래밍 언어의 핵심도 비슷합니다. 자바스크립트는 데이터를 저장하는 방법이 다양하며, 이 중 하나의 중요한 개념은 변수입니다.

변수는 데이터를 담아두는 공간으로, 그 값을 나중에 사용하기 위해 저장합니다. 간단히 말해, 자바스크립트에서 변수는 정보를 담아둔 상자라고 볼 수 있습니다. 이러한 변수 개념은 프로그래밍의 기본이며, 데이터를 다루는 핵심 도구입니다.

데이터를 저장하고 활용하는 과정을 자세히 이해하는 것은 프로그래밍을 배우는데 필수적입니다. 자바스크립트에서 변수를 사용하여 데이터를 저장하는 방법을 이해하면, 이후에 조건문, 반복문, 함수 등을 통해 데이터를 더욱 효과적으로 다룰 수 있게 됩니다. 이것이 프로그래밍의 기초이며, 더 나아가서 웹 애플리케이션을 개발하거나 복잡한 프로그램을 작성할 때에도 핵심적인 역할을 하게 됩니다.

01. 변수 : 데이터 저장하기

변수를 정의하고 값을 할당하는 예제입니다. 

{
        let x = 100;            
        let y = 200;            
        let z = "javascript";   

        console.log(x);     
        console.log(y);     
        console.log(z);     

        // 100
        // 200
        // javascript
}

변수를 사용하면 값을 저장하고 필요할 때 참조하여 사용할 수 있습니다. 이러한 변수 개념을 이해하면 프로그래밍에서 데이터의 저장과 활용 방법을 이해하는 데 큰 도움이 됩니다.

  • let x = 100; → 변수 x를 선언하고, 값 100을 할당합니다. let 키워드는 블록 범위의 지역 변수를 선언하는데 사용됩니다. 여기서 x는 숫자 100을 담는 변수입니다.
  • let y = 200; → 변수 y를 선언하고, 값 200을 할당합니다. 이는 변수 x와 같은 방식으로 동작합니다. y는 숫자 200을 담는 변수입니다.
  • let z = "javascript"; → 변수 z를 선언하고, 문자열 "javascript"을 할당합니다. 자바스크립트에서 문자열은 큰따옴표 또는 작은따옴표로 감싸서 표현합니다. z는 문자열 "javascript"을 담는 변수입니다.

02. 변수 : 데이터 저장하기 + 데이터 변경하기

변수의 값을 변경하는 예제입니다. 변수는 데이터를 저장할 수도 있지만, 필요한 경우 변경도 가능합니다. 

{
        let x = 100;           
        let y = 200;            
        let z = "javascript";   

        x = 300;        
        y = 400;       
        z = "jquery";   

        console.log(x);   
        console.log(y);   
        console.log(z);   

        // 300
        // 400
        // jquery
}

변수의 값을 변경하여 프로그램의 상태를 동적으로 조작할 수 있습니다.

  • let x = 100; → 변수 x를 선언하고, 값 100을 할당합니다.
  • let y = 200; → 변수 y를 선언하고, 값 200을 할당합니다.
  • let z = "javascript"; → 변수 z를 선언하고, 문자열 "javascript"을 할당합니다.
  • x = 300; → 변수 x의 값을 300으로 변경합니다. 이전에 선언한 x 변수의 값 100은 300으로 덮어씌워집니다.
  • y = 400; → 변수 y의 값을 400으로 변경합니다. 이전에 선언한 y 변수의 값 200은 400으로 덮어씌워집니다.
  • z = "jquery"; → 변수 z의 값을 문자열 "javascript"에서 "jquery"로 변경합니다.

03. 변수 : 데이터 저장하기 + 데이터 변경하기 + 데이터 추가하기

변수에 값을 추가하는 예제입니다. 변수는 데이터를 저장하고, 변경하고, 추가할 수 있습니다. 

{
        let x = 100;           
        let y = 200;           
        let z = "javascript";  

        x += 300;       
        y += 400;       
        z += "jquery";   

        console.log(x);   
        console.log(y);    
        console.log(z);   
        
        // 400
        // 600
        // javascriptjquery
}

변수 x에 300을 추가하여 400이, 변수 y에 400을 추가하여 600이, 그리고 변수 z에 "jquery"를 추가하여 "javascriptjquery"가 되어 출력됩니다. 이는 변수에 값을 누적하거나 연결하는 데 사용되는 흔한 패턴입니다.

  • let x = 100; → 변수 x를 선언하고, 값 100을 할당합니다.
  • let y = 200; → 변수 y를 선언하고, 값 200을 할당합니다.
  • let z = "javascript"; → 변수 z를 선언하고, 문자열 "javascript"을 할당합니다.
  • x += 300; → 변수 x의 값에 300을 추가합니다. 이는 변수 x의 현재 값 100에 300을 더한 값 400을 저장합니다.
  • y += 400; → 변수 y의 값에 400을 추가합니다. 이는 변수 y의 현재 값 200에 400을 더한 값 600을 저장합니다.
  • z += "jquery"; → 변수 z의 값에 문자열 "jquery"를 추가합니다. 이는 변수 z의 현재 값 "javascript"에 "jquery"를 연결한 값 "javascriptjquery"을 저장합니다.

04. 상수 : 데이터 저장하기 + 데이터 변경하기(X)

변수와 상수는 데이터를 저장하는 것은 동일하지만, 상수는 데이터를 변경할 수 없습니다. 

{
        const x = 100;            
        const y = 200;           
        const z = "javascript";   

        // x = 300;        //주석 표시가 없으면 에러가 발생합니다.
        // y = 400;        
        // z = "jquery";   

        console.log(x);     
        console.log(y);     
        console.log(z);    

        //100
        //200
        //javascript
}

const로 선언된 상수는 한 번 할당되면 값이 변경될 수 없습니다. 따라서 주석 처리된 부분에서 에러가 발생합니다. 상수는 보통 변경되지 않아야 하는 값을 나타낼 때 사용됩니다.

  • const x = 100; → 상수 x를 선언하고, 값 100을 할당합니다. const 키워드는 변수와 유사하지만, 한 번 할당된 값은 변경할 수 없습니다.
  • const y = 200; → 상수 y를 선언하고, 값 200을 할당합니다. y도 x와 같은 방식으로 선언됩니다.
  • const z = "javascript"; → 상수 z를 선언하고, 문자열 "javascript"을 할당합니다. z 역시 x와 y와 같은 방식으로 정의됩니다.
  • 주석 처리된 줄들 (// x = 300;, // y = 400;, // z = "jquery";)은 상수를 변경하려는 시도를 보여주는 부분입니다. 하지만 const로 선언된 상수의 값은 변경할 수 없기 때문에 에러가 발생합니다. 이 줄들을 주석 해제하면 에러가 발생합니다.

05. 배열 : 데이터 저장하기(여러개) : 표현방법1 : 배열 선언

변수는 하나의 데이터를 저장할 수 있지만, 배열은 여러개의 데이터를 순차적으로 저장할 수 있습니다. 다음은 배열을 생성하고 배열의 원소에 값을 할당한 후, 해당 값을 출력하는 예제입니다 

{
        const arr = new Array();
        
        arr[0] = 100;
        arr[1] = 200;
        arr[2] = "javascript";

        console.log(arr[0]);
        console.log(arr[1]);
        console.log(arr[2]);

        // 100
        // 200
        // javascript
}

배열 arr에 각각의 값을 할당하고 해당 값을 출력하는 예제입니다. 배열은 여러 개의 값들을 순서대로 담는 자료구조로 사용되며, 인덱스를 사용하여 각 원소에 접근할 수 있습니다.

  • const arr = new Array(); → arr라는 이름의 배열을 생성합니다. new Array()는 빈 배열을 생성하는 표현식입니다. 이 배열은 상수 arr에 할당됩니다.
  • arr[0] = 100; → 배열 arr의 첫 번째 원소에 값 100을 할당합니다. 배열의 인덱스는 0부터 시작하므로 첫 번째 원소는 인덱스 0에 해당합니다.
  • arr[1] = 200; → 배열 arr의 두 번째 원소에 값 200을 할당합니다. 두 번째 원소의 인덱스는 1입니다.
  • arr[2] = "javascript"; → 배열 arr의 세 번째 원소에 문자열 "javascript"을 할당합니다. 세 번째 원소의 인덱스는 2입니다.

06. 배열 : 데이터 저장하기(여러개) : 표현방법2 : 배열 선언과 동시에 초깃값 설정

배열을 생성하고 초기값을 설정한 후, 배열의 원소에 접근하여 값을 출력하는 예제입니다 

{
        const arr = new Array(100, 200, "javascript");

        console.log(arr[0]);
        console.log(arr[1]);
        console.log(arr[2]);

        // 100
        // 200
        // javascript
}

배열 arr은 초기값으로 100, 200, "javascript"을 가지고 있으며, 각 원소에 접근하여 해당 값을 출력하는 예제입니다. 이렇게 생성자를 사용하여 배열을 생성하면 한 줄로 여러 개의 값을 설정할 수 있습니다.

  • const arr = new Array(100, 200, "javascript"); → arr라는 이름의 배열을 생성하면서 초기값 100, 200, "javascript"을 전달합니다. 이렇게 생성된 배열은 세 개의 원소를 가지며, 각각 숫자 100, 200, 문자열 "javascript"가 됩니다.

07. 배열 : 데이터 저장하기(여러개) : 표현방법3 : 리터럴 방식으로 배열 선언

빈 배열을 생성하고 배열의 원소에 값을 할당한 후, 해당 값을 출력하는 예제입니다. new Array를 선언하지 않고, [ ]를 사용하는 것이 기존과 다릅니다. 

{
        const arr = [];

        arr[0] = 100;
        arr[1] = 200;
        arr[2] = "javascript";

        console.log(arr[0]);
        console.log(arr[1]);
        console.log(arr[2]);

        // 100
        // 200
        // javascript
}

빈 배열 arr을 생성하고 각각의 인덱스에 해당하는 원소에 값을 할당한 후, 해당 값을 출력하는 예제입니다. 배열은 여러 값을 저장하는 자료구조로, 인덱스를 사용하여 원소에 접근하거나 값을 할당할 수 있습니다.

  • const arr = []; → arr라는 이름의 빈 배열을 생성합니다. 대괄호 []를 사용하여 빈 배열을 생성하였습니다. 이 배열은 상수 arr에 할당됩니다.
  • arr[0] = 100; → 배열 arr의 첫 번째 원소에 값 100을 할당합니다. 이렇게 인덱스를 사용하여 배열의 원소에 값을 할당할 수 있습니다.

11. 배열 : 데이터 저장(여러개) : 표현방법7 : 배열 구조 분해 할당

배열 구조 분해 할당(Array Destructuring) 기능을 사용하여 배열의 요소를 개별 변수에 할당하는 예시입니다. 

{
        const arr = [100, 200, "javascript"];

        const [a, b, c] = arr;

        console.log(a);
        console.log(b);
        console.log(c);

        // 100
        // 200
        // javascript
}

배열 구조 분해 할당을 사용하면 배열의 요소를 각각의 변수로 빠르고 간편하게 할당할 수 있으며, 코드의 가독성을 높일 수 있습니다.

  • const arr → arr이라는 이름의 상수를 선언하고 있습니다. 이 변수는 배열을 가리키는 참조를 저장합니다.
  • [100, 200, "javascript"] → 배열은 대괄호 []로 둘러싸인 값들의 목록입니다. 이 배열은 세 개의 요소를 가지고 있습니다. 첫 번째 요소는 100, 두 번째 요소는 200, 세 번째 요소는 문자열 "javascript"입니다.
  • const [a, b, c] → 이 부분은 배열 해체를 사용하여 arr 배열의 요소를 개별 변수에 할당하고 있습니다. 대괄호 [] 안에 있는 변수 이름들 [a, b, c]은 배열 arr의 순서대로 요소들을 차례대로 할당할 변수 이름을 나타냅니다.
  • arr → 할당할 대상인 배열입니다.

2차 배열에서도 똑같이 사용할 수 있습니다. 

{
        const arr = [100, 200, ["javascript", "jquery"]];

        const [a, b [x, y]] = arr;

        console.log(a);
        console.log(b);
        console.log(x);
        console.log(y);

        // 100
        // 200
        // javascript
        // jquery
}

2차 배열 배열 구조 분해 할당을 사용한 예제입니다.

  • arr이라는 배열을 선언하고 초기화합니다. 이 배열은 [100, 200, ["javascript", "jquery"]]와 같은 구조를 가지고 있습니다.
  • a에는 arr 배열의 첫 번째 요소인 100이 할당됩니다.
  • b에는 arr 배열의 두 번째 요소인 200이 할당됩니다.
  • [x, y]는 arr 배열의 세 번째 요소인 ["javascript", "jquery"]를 해체합니다. x에는 "javascript"가 할당되고, y에는 "jquery"가 할당됩니다.

12. 배열 : 데이터 저장(여러개) : 표현방법8 : 배열 펼침 연산자

배열 펼침 연산자(Array Spread Operator)는 배열의 요소들을 펼쳐서 개별 값으로 가져오거나, 배열을 병합할 때 사용되는 JavaScript의 기능입니다. ...기호를 사용하여 배열을 펼칠 수 있습니다. 이를 통해 배열을 개별 값으로 분리하거나, 다른 배열에 요소를 병합하는 등의 작업을 쉽게 수행할 수 있습니다. 

{
        const arr1 = [100, 200, "javascript"];
        const arr2 = [300, 400, "jquery"];
        const arr3 = [...arr1, 500];

        console.log(arr1);              
        console.log(...arr1);           
        console.log(...arr1, ...arr2);  
        console.log(...arr3);  
        
        // 100, 200, javascript
        // 100 200 javascript
        // 100 200 javascript 300 400 jquery
        // 100 200 javascript 500
}

펼침 연산자를 사용하면 배열을 개별 값으로 펼치거나 다른 배열에 병합하는 등의 다양한 작업을 할 수 있습니다.

  • const arr1 → arr1이라는 이름의 상수를 선언하고 있습니다. 이 변수에는 [100, 200, "javascript"]라는 배열이 할당됩니다.
  • const arr2 → arr2이라는 이름의 상수를 선언하고 있습니다. 이 변수에는 [300, 400, "jquery"]라는 배열이 할당됩니다.
  • const arr3 → arr3이라는 이름의 상수를 선언하고 있습니다. 이 변수에는 arr1 배열의 요소들과 추가로 500이라는 숫자가 포함된 배열이 할당됩니다. 스프레드 연산자(...)를 사용하여 배열 arr1의 요소들을 가져온 뒤, 500을 추가하여 새 배열을 생성합니다.
  • console.log(arr1) → arr1 배열을 그대로 출력합니다. 따라서 출력 결과는 [100, 200, "javascript"]가 됩니다.
  • console.log(...arr1) → 배열 arr1의 요소를 스프레드 연산자를 사용하여 개별적인 값으로 펼쳐서 출력합니다. 따라서 출력 결과는 100 200 javascript가 됩니다.
  • console.log(...arr1, ...arr2) → 배열 arr1의 요소와 배열 arr2의 요소를 스프레드 연산자를 사용하여 모두 펼쳐서 출력합니다. 따라서 출력 결과는 100 200 javascript 300 400 jquery가 됩니다.
  • console.log(...arr3) → 배열 arr3의 요소를 스프레드 연산자를 사용하여 개별적인 값으로 펼쳐서 출력합니다. arr3는 [100, 200, "javascript", 500]이므로 출력 결과는 100 200 javascript 500가 됩니다.

13. 객체 : 데이터 저장(키와값) : 표현방법1 : 객체 선언 후 배열 방식으로 데이터 입력

객체도 변수와 배열과 같이 데이터를 저장하는 저장소 역할을 합니다. 변수와 배열과 다르게 객체는 키와 값으로 데이터를 저장할 수 있습니다. 다음의 예제는 객체를 선언하고, 배열 방식을 이용하여 데이터를 저장하는 예제입니다.

  • const obj = new Object(); → obj라는 이름의 빈 객체를 생성합니다. new Object()를 사용하여 빈 객체를 생성하고, 이 객체는 상수 obj에 할당됩니다.

  • obj[1] = 200; → 객체 obj의 속성 1에 값 200을 할당합니다. 마찬가지로 숫자 1도 객체의 속성 이름으로 사용됩니다.
  • obj[2] = "javascript"; → 객체 obj의 속성 2에 문자열 "javascript"을 할당합니다. 숫자 2를 속성 이름으로 사용하여 해당 속성에 값을 할당합니다.

++ 객체 좀 더 알아보기 ++

객체(Object)는 프로그래밍에서 데이터와 해당 데이터에 적용되는 동작(메서드)을 그룹화한 것으로, 데이터와 기능을 하나의 단위로 묶어 표현하는 자료구조입니다. 객체는 다양한 유형의 데이터와 그에 관련된 동작을 한 곳에 모아서 표현함으로써 프로그램을 조직화하고 구조화하는 데 도움을 줍니다.

객체는 여러 속성과 메서드를 하나의 단위로 묶어서 사용하므로, 코드의 가독성과 재사용성을 높이는 데 도움이 됩니다. 자바스크립트를 포함한 다양한 프로그래밍 언어에서 객체 지향 프로그래밍(Object-Oriented Programming, OOP)의 핵심 개념 중 하나입니다. 객체는 현실 세계의 실체를 모델링하고 프로그램 내에서 효율적으로 다룰 수 있게 도와주는 중요한 도구입니다.

  • 속성 (Property): 객체의 속성은 데이터 값을 나타냅니다. 속성은 이름(키)과 값으로 구성되며, 이름은 일반적으로 문자열로 표현됩니다. 속성은 다양한 유형의 데이터를 포함할 수 있습니다. 예를 들어, 객체가 사람을 나타낼 경우 이름, 나이, 주소 등이 해당 객체의 속성이 될 수 있습니다.
  • 메서드 (Method): 객체의 메서드는 객체가 수행할 수 있는 동작을 정의합니다. 메서드는 함수의 형태를 가지며, 해당 객체와 관련된 작업을 수행합니다. 예를 들어, 객체가 자동차를 나타낸다면 주행, 멈춤, 가속 등의 동작을 메서드로 정의할 수 있습니다.

초기값을 가진 객체 obj의 각 속성에 접근하여 해당 값을 출력하는 예제입니다. 객체는 중괄호 내에 속성과 값을 나열하여 생성할 수 있으며, 속성에는 문자열이나 숫자를 사용할 수 있습니다.

결과적으로, 객체 obj의 속성과 메서드를 활용하여 값을 출력하고 함수를 실행하는 예제입니다. 객체는 다양한 유형의 데이터와 동작을 함께 묶어서 사용할 수 있으며, 메서드 내에서 this 키워드를 사용하여 객체 내의 다른 속성에 접근할 수 있습니다.

  • const obj = {...} → obj라는 이름의 객체를 생성합니다. 객체 내에는 a, b, c, d, e, f라는 다양한 속성과 해당 값을 가진 메서드가 포함되어 있습니다.

  • obj.e(); → 객체 obj의 e 속성으로 정의된 함수를 실행합니다. 이 함수는 obj.c 속성 값을 사용하여 "javascript가 실행되었습니다."를 출력합니다.
  • obj.f(); → 객체 obj의 f 속성으로 정의된 함수를 실행합니다. 이 함수는 this.c를 사용하여 obj.c 속성 값을 사용하여 "javascript가 실행되었습니다."를 출력합니다.

++ 객체 구조 분해 할당 좀 더 알아보기 ++

객체 구조 분해 할당(Object Destructuring Assignment)은 JavaScript에서 객체의 속성을 변수로 분해하여 가져오는 기능입니다. 이를 통해 객체의 각 속성 값을 개별 변수에 할당할 수 있습니다. 객체 구조 분해 할당은 ES6(ECMAScript 2015)에서 도입된 기능으로, 코드의 가독성과 유지 보수성을 높이는 데 도움이 됩니다. 객체 구조 분해 할당을 사용하면

  • 코드가 간결해지고 가독성이 높아집니다.
  • 원하는 속성만 선택적으로 가져와서 사용할 수 있습니다.
  • 중첩된 객체에서도 내부 속성에 접근이 용이해집니다.
  • 기본값을 설정하거나 속성 이름을 변경하여 사용할 수 있습니다.

22. 객체 : 데이터 저장(키와값) : 표현방법10 : 객체 펼침 연산자

객체와 객체 펼침 연산자를 사용하여 객체를 생성하고 활용하는 방법을 보여주는 예시입니다 

{
        const obj1 = { a: 100, b: 200, c: "javascript" };
        const obj2 = { ...obj1, d: "jquery" };
        const obj3 = { ...obj1, ...obj2 };

        console.log(obj1);
        console.log(obj2);
        console.log(obj2.b);
        console.log(obj2.c);
        console.log(obj2.d);
        console.log(obj3);
}

객체 펼침 연산자를 통해 속성들이 병합되고 새로운 객체가 생성되는 것을 확인할 수 있습니다.

  • const obj1 → obj1이라는 이름의 상수를 선언하고 있습니다. 이 변수에는 객체 { a: 100, b: 200, c: "javascript" }가 할당됩니다. 이 객체에는 a, b, c라는 세 개의 속성(key)이 포함되어 있습니다.
  • const obj2 → obj1 객체의 속성들을 모두 펼쳐서 새로운 객체 { ...obj1 }를 생성하고, 이 객체에 d 속성을 추가하여 jquery 값을 할당합니다. 따라서 obj2 객체는 { a: 100, b: 200, c: "javascript", d: "jquery" }가 됩니다.
  • const obj3 → obj1과 obj2 객체의 속성들을 모두 펼쳐서 새로운 객체 { ...obj1, ...obj2 }를 생성합니다. 이렇게 하면 obj1과 obj2의 속성들이 결합됩니다. 따라서 obj3 객체는 { a: 100, b: 200, c: "javascript", d: "jquery" }가 됩니다.
  • console.log(obj1); → obj1 객체를 그대로 출력합니다. 출력 결과는 { a: 100, b: 200, c: "javascript" }가 됩니다.
  • console.log(obj2); → obj2 객체를 그대로 출력합니다. 출력 결과는 { a: 100, b: 200, c: "javascript", d: "jquery" }가 됩니다.
  • console.log(obj2.b); → obj2 객체의 b 속성에 접근하여 해당 값을 출력합니다. 출력 결과는 200이 됩니다.
  • console.log(obj2.c); → obj2 객체의 c 속성에 접근하여 해당 값을 출력합니다. 출력 결과는 "javascript"가 됩니다.
  • console.log(obj2.d); → obj2 객체의 d 속성에 접근하여 해당 값을 출력합니다. 출력 결과는 "jquery"가 됩니다.
  • console.log(obj3); → obj3 객체를 그대로 출력합니다. 출력 결과는 { a: 100, b: 200, c: "javascript", d: "jquery" }가 됩니다.

++ 객체 펼침 연산자 좀 더 알아보기 ++

객체 펼침 연산자 (Object Spread Operator)는 객체의 속성들을 펼쳐서 다른 객체에 복사하거나 새로운 속성을 추가할 때 사용되는 JavaScript의 기능입니다. ... 기호를 사용하여 객체의 속성들을 펼칠 수 있습니다. 이를 통해 객체를 병합하거나 새로운 객체를 생성하는 데에 편리하게 사용할 수 있습니다.

  • 객체의 속성을 펼치기: 객체 펼침 연산자를 사용하여 객체의 각 속성을 개별적인 값으로 가져옵니다.
  • 다른 객체와 병합: 객체 펼침 연산자를 사용하여 하나의 객체의 속성들을 다른 객체에 병합하거나 새로운 속성을 추가할 수 있습니다.
  • 새로운 객체 생성: 객체 펼침 연산자를 사용하여 기존 객체를 기반으로 속성을 추가, 수정하거나 덮어쓴 새로운 객체를 생성합니다.

23. 파일 : 데이터 교환(JSON)

JSON(JavaScript Object Notation)은 데이터 교환 형식의 하나로, 텍스트 형태로 표현되며, 데이터를 구조화하고 표현하기 위한 경량의 형식입니다. JSON은 웹 애플리케이션에서 데이터를 전송하거나 저장하는 데 주로 사용됩니다. 

{
        // 01 JSON 데이터를 객체로 변환
        const jsonString = '{"name": "John", "age": 30}';
        const jsonObject = JSON.parse(jsonString);
        console.log(jsonObject.name); 
        
        // "John"

        // 02 객체를 JSON 형식의 문자열로 변환
        var person = { name: "John", age: 30 };
        var jsonString = JSON.stringify(person);
        console.log(jsonString); 
        
        // '{"name":"John","age":30}'
}

데이터를 JSON 형식에서 JavaScript 객체로 가져오거나, JavaScript 객체를 JSON 형식으로 다시 돌려보낼 때 유용합니다. 이를 통해 웹 애플리케이션과 서버 간에 데이터를 교환하거나 데이터를 저장하는 데 사용할 수 있습니다. JSON은 데이터 교환 형식으로 널리 사용되며, JavaScript의 JSON.parse()와 JSON.stringify() 함수는 JSON 데이터를 처리하는 데 중요한 역할을 합니다.

  • 01 → JSON 형식의 문자열인 jsonString을 JavaScript 객체로 변환하는 과정을 보여줍니다. JSON.parse() 함수를 사용하여 JSON 문자열을 파싱하고, 그 결과를 jsonObject 변수에 저장합니다. 그리고나서 jsonObject.name을 사용하여 객체에서 이름을 가져와서 콘솔에 출력합니다. 출력 결과는 "John"이 됩니다.
  • 02 → JavaScript 객체 person을 JSON 형식의 문자열로 변환하는 과정을 보여줍니다. JSON.stringify() 함수를 사용하여 JavaScript 객체를 JSON 형식의 문자열로 직렬화(serialization)합니다. 그 결과를 jsonString 변수에 저장하고, console.log(jsonString)을 통해 JSON 문자열을 콘솔에 출력합니다. 출력 결과는 {"name":"John","age":30}이 됩니다.
